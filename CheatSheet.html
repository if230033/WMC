Basic Web Techniques Cheat Sheet

1. World Wide Web (WWW)

Three Essential Technologies: According to the history of the World Wide Web, there are three key technologies:

  • HTML (HyperText Markup Language): The standard language for creating web pages and web applications.
  • URI (Uniform Resource Identifier): A way to identify resources on the web, commonly in the form of a URL (Uniform Resource Locator).
  • HTTP (Hypertext Transfer Protocol): The protocol used to transfer web pages from a server to a browser.

2. URL (Uniform Resource Locator)

URL Syntax: A URL typically consists of:

  • Scheme: Identifies the protocol (e.g., http, https, ftp).
  • Host: The domain name or IP address of the web server (e.g., www.example.com).
  • Path: The file location or resource on the server (e.g., /index.html).
  • Query: Additional parameters (optional), often used in dynamic web pages.

3. Web Server

Web Server Overview: A web server is a system that delivers web pages to users based on their HTTP requests. It stores and processes files like HTML documents and media.

4. Web Browser

Function of a Web Browser: A web browser fetches content from web servers (using HTTP/HTTPS), interprets HTML/CSS/JavaScript, and renders the web page for the user.

5. HTML Elements

DOCTYPE <!DOCTYPE html>: Declares the document type and version of HTML.

html <html></html>: Root element of an HTML document.

head <head></head>: Contains meta information about the document (e.g., character encoding, title).

title <title></title>: Sets the title of the webpage, displayed on the browser tab.

meta <meta charset="UTF-8">: Declares the character encoding for the document.

body <body></body>: The main content of the webpage is placed here.

strong <strong></strong>: Defines important text, typically displayed in bold.

em <em></em>: Defines emphasized text, typically displayed in italics.
