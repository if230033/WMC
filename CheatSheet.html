Basic Web Techniques Cheat Sheet

1. World Wide Web (WWW)

Three Essential Technologies: HTML, HTTP, URLs, hypertext navigation, and client-server architecture made the World Wide Web possible, leading to the sharing of information globally.

Addressing Resources: Resources like HTML pages and images are addressed using URLs. A URL specifies the protocol, domain, and path to the resource on a server.

General Components of a URL:

  • Scheme: Specifies the protocol (e.g., http or https).
  • Domain: The web address (e.g., www.example.com).
  • Path: The location of the file or resource on the server.
  • Port: Optional part specifying the server's port.
  • Query/Fragment: Optional parameters or fragment identifiers.

2. Web Server

What is a Web Server? A web server is a computer system that hosts websites and delivers content to users over the internet.

Web Server Function: It delivers web content to web browsers when requested via HTTP or HTTPS protocols.

Storing Content: Content such as HTML files, CSS sheets, and images are stored in the server's file system and accessed using a URL when requested by a user.

Describing Webpage Content: Content is described using HTML elements, metadata, CSS for styling, and JavaScript for interactivity.

3. Web Client

What is a Web Client? A web client, typically a web browser, communicates with web servers to fetch and display content from the internet.

Web Client Function: It sends requests to web servers and processes responses (e.g., HTML) to display content to the user.

Web Client vs. Web Browser: A web browser is a type of web client used for displaying web content. Not all web clients are web browsers, but all browsers are web clients.

4. Hello World

The <head> Element: The head element contains metadata about the document, which is not directly displayed on the webpage.

<!DOCTYPE html> Declaration: This is not an HTML tag, but a declaration that tells the browser what type of document it is. It is required for all HTML documents.

Top Most Elements Inside <html>: The two top elements inside the HTML element are <head> and <body>.

Emphasizing Text: Text can be emphasized using the <em> element, which is displayed in italics in the browser.

Marking Important Text: The <strong> element is used to mark important text, which is displayed in bold in the browser.

HTML and CSS Cheat Sheet

HTML Basics

HTML Basics: The foundational structure of an HTML document consists of elements like <html>, <head>, and <body>. The document starts with a <!DOCTYPE html> declaration.

HTML Elements

Elements: HTML elements consist of opening and closing tags (e.g., <p> and </p>) and content between them. Some elements, like images, are void elements, meaning they don't have closing tags.

HTML Attributes

Attributes: HTML attributes provide additional information about elements. For example, <a href="url"> specifies the link's destination.

Headings

Headings: HTML provides six levels of headings, from <h1> (the most important) to <h6> (the least important).

Paragraphs and Line Breaks

Paragraphs: Use the <p> element to create paragraphs. Avoid excessive use of <br> for line breaks; use paragraphs to create clean separations between text blocks.

Comments

Comments: Comments in HTML are written between <!-- --> and are not displayed in the browser.

Links

Links: Use the <a> element to create links. Example: <a href="https://example.com">Visit Example</a>.

Images

Images: Use the <img> element to display images. Include the alt attribute to provide alternate text for accessibility and SEO. Example: <img src="image.jpg" alt="Description of image">.

Head Element

Head Element: The <head> element contains metadata such as the title, character encoding, and other information. Example: <meta charset="UTF-8">.

CSS Basics

CSS Syntax: A CSS rule consists of a selector and a declaration block. Example:

p {
  color: blue;
  font-size: 16px;
}

CSS Colors

Colors: Colors can be defined using names (e.g., blue), hexadecimal codes (e.g., #0000FF), or RGB values.

CSS Backgrounds

  • Background Color: Define a background color using background-color.
  • Background Image: Set a background image with background-image: url('image.jpg'). You can control its repetition and positioning.

CSS Text Properties

  • Text Color: Set text color with color.
  • Text Alignment: Align text using text-align (e.g., left, right, center).
  • Text Decoration: Use text-decoration for underline, overline, etc.
  • Text Transformation: Transform text to uppercase, lowercase, or capitalize using text-transform.

CSS Fonts

  • Font Family: Define font family using font-family, such as sans-serif or serif.
  • Font Size: Set font size using pixels (e.g., 16px) or relative units (e.g., 1em).
  • Font Weight: Define the thickness of text using font-weight.

Styling Links

Link Styling: You can style links using pseudo-classes like :hover, :visited, and :active.

CSS Combinators

Combinators: CSS combinators like descendant (div p), child (div > p), and sibling (h1 + p) allow more specific targeting of elements.

Pseudo-Classes

  • Anchor Pseudo-Classes: :link, :visited, :hover, :active for styling link states.
  • :first-child: Target the first child element of its parent using :first-child.
